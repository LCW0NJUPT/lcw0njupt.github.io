小米路由器docker怎么使用，换源方法

 

1、ssh登录路由器，插入64G以上存储设备，安装docker，安装第三方管理（simple-docker）

2、修改/tmp/dockerd/daemon.json如下

 

"registry-mirrors": [ "https:\/\/docker.1ms.run", "https:\/\/docker.xuanyuan.me", "https:\/\/docker.xiaoqi.online", "https:\/\/docker.zhai.cm", "https:\/\/hub.rat.dev" ],

 

注意这里的镜像站时刻会变会挂，以上截至2025.2.1尚能使用

 

 

3、 ps | grep dockerd 查看进程

kill了有json的那条的进程号

kill -9 18926

等待进程重启，此时已经可以正常pull docker镜像了
